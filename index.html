Projects

  • Developed a human sensing system using Wi-Fi CSI data collected via ESP32 devices. Built and trained a BiLSTM model to detect and classify human presence and movement patterns without the need for cameras, preserving user privacy. Tested the robustness of the dataset using an LSTM baseline model and improved overall performance through an overlapping segment scoring technique, which enhanced temporal resolution and model accuracy.
  • Developed a smart navigation system using Raspberry Pi to assist visually impaired users with real-time obstacle detection and audio guidance. The system combines ultrasonic sensors, a GPS module, and Google Directions API to identify obstacles and determine location. Text-to-speech (TTS) software converts this data into spoken alerts delivered through headphones or a speaker. An SMS alert feature, integrated via ClickSend, enables emergency communication. The Raspberry Pi serves as the main controller, processing all sensor inputs to provide accurate, responsive navigation support.
  • Designed an infrared-based security system using an IR transmitter-receiver pair, LM358 op-amp, and 555 Timer IC to detect intrusions. When the IR beam is interrupted, the system triggers an audible alarm, offering a simple and cost-effective solution for perimeter protection.
